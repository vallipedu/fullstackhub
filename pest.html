Pest Management System for Farmers

This project is about building a simple web app to help farmers identify and manage common crop pests in Andhra Pradesh.

Problem Statement

Pests can ruin a farmer's crop and cause huge losses. Your task is to design and develop a Full_Stack Application that gives a list of suitable pesticides based on the crop and the pest attacking it. The application must use a more complex data structure, with multiple fields for each recommendation.

Functional Requirements

  • Input: The app should have two dropdown lists. One for the crop and one for the pest. The user must select both.
  • Recommendation: When the user selects a crop and a pest, the app must show the recommended pesticide and any associated notes.
  • User Interface: The interface should be simple and easy to use. It needs two dropdowns and a dedicated area to show the result.

Data

Use the following structured data. Notice that each pest has its own object with `pesticide` and `notes` fields. 

const pestData = {
    "Rice": {
        "Rice Stem Borer": { pesticide: "Chlorantraniliprole", notes: "Apply at vegetative stage. Effective on young larvae." },
        "Rice Hispa": { pesticide: "Monocrotophos", notes: "Use during early stages of infestation. Highly effective." }
    },
    "Groundnut": {
        "Leaf Miner": { pesticide: "Chlorpyrifos", notes: "Broad-spectrum insecticide. Apply as foliar spray." },
        "Aphids": { pesticide: "Imidacloprid", notes: "Systemic insecticide. Effective against sucking pests." }
    },
    "Sugarcane": {
        "Early Shoot Borer": { pesticide: "Carbofuran", notes: "Granular insecticide for soil application at planting." },
        "White Grubs": { pesticide: "Phorate", notes: "Apply in the soil to control grubs. Use with caution." }
    },
    "Cotton": {
        "Cotton Bollworm": { pesticide: "Pyrethroids", notes: "Commonly used. Rotate with other classes to prevent resistance." },
        "Whitefly": { pesticide: "Acetamiprid", notes: "Neonicotinoid insecticide. Safe for beneficial insects." }
    },
    "Maize": {
        "Fall Armyworm": { pesticide: "Emamectin benzoate", notes: "Use when infestation is severe. Highly potent." },
        "Maize Stem Borer": { pesticide: "Spinosad", notes: "Natural product insecticide. Target the stem base." }
    }
};

Testing Requirements

Test your application with these cases:

  • Test Case 1.1: Correct Recommendation
    • Given: The user selects "Groundnut" and "Leaf Miner".
    • Expected Result: The application must show the recommended pesticide: "Chlorpyrifos" and the note: "Broad-spectrum insecticide. Apply as foliar spray."
  • Test Case 1.2: Missing Selection
    • Given: The user selects a crop but not a pest.
    • Expected Result: A message box should pop up, asking the user to make a full selection.
  • Test Case 1.3: No Data Available
    • Given: The user selects a combination that doesn't have a match (e.g., "Maize" and "White Grubs").
    • Expected Result: A message box should indicate that no recommendation is available for that combination.
