Fertilizer Recommendation for Indian Farmers

This document outlines the problem and functional requirements for a web application designed to assist Indian farmers in selecting the correct fertilizer for their crops based on soil type.

Problem Statement

The optimal use of fertilizers is crucial for maximizing crop yield while maintaining soil health. The inappropriate application of fertilizers can lead to soil degradation and reduce agricultural productivity. The objective is to design and develop a Full_Stack Application that recommends a list of suitable fertilizers to farmers based on the combination of their soil type and the crop they intend to grow.

Functional Requirements

  • Input: The system must allow the user to select a soil type and a specific crop from two separate predefined lists.
  • Recommendation: Based on the selected soil-crop combination, the application must display a list of suitable fertilizers. If no recommendation exists for a particular combination, an appropriate message should be shown.
  • User Interface: The interface must be straightforward and intuitive, consisting of two selectors (one for soil, one for crop) and a dedicated section to display the recommended fertilizers.

Data

The following data structure will be used to map soil and crop combinations to suitable fertilizers: 

const fertilizerData = {
    Alluvial: {
        Rice: ["Urea", "DAP"],
        Wheat: ["NPK", "Urea"],
        Sugarcane: ["Urea", "Potash"],
    },
    Black: {
        Cotton: ["DAP", "Urea"],
        Sugarcane: ["Potash", "Urea"],
        Jowar: ["NPK"],
    },
    Red: {
        Groundnuts: ["Gypsum", "DAP"],
        Maize: ["Urea", "NPK"],
    },
    Laterite: {
        Tea: ["Ammonium Sulphate", "Potash"],
        Coffee: ["DAP", "Urea"],
    },
    Desert: {
        Bajra: ["NPK"],
    }
};

Testing Requirements

The application's functionality must be verified with the following test cases:

  • Test Case 1.1: Valid Selection
    • Given: The user selects a valid soil and crop combination (e.g., "Alluvial Soil" and "Rice").
    • Expected Result: The application must correctly display the list of recommended fertilizers.
  • Test Case 1.2: Incomplete Selection
    • Given: The user selects only a soil type but no crop, or vice-versa.
    • Expected Result: An informative message box is displayed, prompting the user to make a complete selection.
  • Test Case 1.3: No Recommendation Available
    • Given: The user selects a combination for which there is no data (e.g., "Laterite Soil" and "Sugarcane").
    • Expected Result: The application displays a message indicating that no fertilizer recommendation is available for the selected combination.
