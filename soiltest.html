Crop Recommendation for Indian Farmers

This document outlines the problem and functional requirements for a web application designed to assist Indian farmers in making informed decisions about crop selection.

Problem Statement

Indian agriculture is heavily dependent on the appropriate selection of crops based on local soil conditions. A lack of specific knowledge can lead to low yields and financial losses. The objective is to design and **develop a Full_Stack Application**, a simple, functional web application that provides crop recommendations to farmers based on the type of soil in their region.

Functional Requirements

  • Input: The system must allow the user (farmer) to select a soil type from a predefined list of common Indian soil types.
  • Recommendation: Based on the selected soil type, the application must display a list of suitable crops that thrive in those conditions.
  • User Interface: The interface must be straightforward and intuitive, consisting of a soil type selector and a dedicated section to display the recommended crops.

Data

The following data structure will be used to map soil types to suitable crops: 

const cropData = {
    Alluvial: ["Rice", "Wheat", "Sugarcane", "Jute", "Tobacco"],
    Black: ["Cotton", "Sugarcane", "Jowar", "Tobacco", "Oilseeds"],
    Red: ["Groundnuts", "Ragi", "Maize", "Tobacco", "Millets"],
    Laterite: ["Tea", "Coffee", "Cashew", "Rubber", "Coconut"],
    Mountain: ["Tea", "Coffee", "Spices", "Fruits (Apples, Pears)", "Maize"],
    Desert: ["Millets", "Barley", "Jowar", "Bajra"]
};

Testing Requirements

The application's functionality must be verified with the following test cases:

  • Test Case 1.1: Valid Soil Selection
    • Given: The user selects a valid soil type (e.g., "Alluvial Soil").
    • Expected Result: The application must correctly display the list of recommended crops associated with that soil type.
  • Test Case 1.2: No Soil Selected
    • Given: The user attempts to get a recommendation without selecting a soil type.
    • Expected Result: An informative message box is displayed, prompting the user to select a soil type.
