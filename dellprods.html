Display List of Products: DELL Computer Services

Problem Statement8

Objective

Design and develop a web application for DELL Computer Services to display product information from a dataset. The system must present the data in a clear, tabular format.

Functional Requirements

  • Display Product List: The system must display a list of products in a tabular format. The quantity of each product should be clearly visible in the table.

Technical Specifications

  • Data Structure: The product data must be implemented as a JavaScript array of objects. Each object will represent a single product and must include properties for id, name, quantity, and price.
  • Sample Records: 
    
[
    {
        id: "D101",
        name: "DELL XPS 13 Laptop",
        quantity: 15,
        price: 120000.00
    },
    {
        id: "D102",
        name: "DELL Inspiron 24 All-in-One",
        quantity: 8,
        price: 75000.00
    },
    {
        id: "D103",
        name: "DELL UltraSharp 27 Monitor",
        quantity: 25,
        price: 35000.00
    }
]
  • Programming Language: JavaScript for all application logic. The solution must make extensive use of modern JavaScript syntax.
  • User Interface: A simple interface built with HTML and a minimal amount of CSS to display the product data in a table.

Testing Requirements

  • Product Display Test:
    • Test Case 1.1: Given a complete product dataset, the application must correctly display all product information in a readable table.
    • Test Case 1.2: The quantity for each product must be displayed correctly as per the data source.
    • Expected Result: A clear, tabular view of the products is displayed, including their quantity.
