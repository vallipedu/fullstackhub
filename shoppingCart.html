DELL Shopping Cart Total Display

Problem Statement9

Objective

Design and develop a web application for DELL that calculates the total cost of a user's shopping cart. The system should process a list of selected products to display the final amount to be paid.

Functional Requirements

  • Calculate Total: The system must calculate and display the total amount a user has to pay for all products in their shopping cart.

Technical Specifications

  • Data Structure: The shopping cart data must be implemented as a JavaScript array of objects. Each object will represent a selected product and must include properties for id, name, description, price, and quantity.
  • Sample Records: 
    
[
    {
        id: "D101",
        name: "DELL XPS 13 Laptop",
        description: "A high-performance ultrabook with a stunning display.",
        price: 120000.00,
        quantity: 1
    },
    {
        id: "D102",
        name: "DELL Inspiron 24 All-in-One",
        description: "A compact desktop solution with a powerful processor.",
        price: 75000.00,
        quantity: 2
    },
    {
        id: "D103",
        name: "DELL UltraSharp 27 Monitor",
        description: "A professional-grade monitor with exceptional color accuracy.",
        price: 35000.00,
        quantity: 1
    }
]
  • Programming Language: JavaScript for all application logic. The solution must make extensive use of modern JavaScript syntax.
  • User Interface: A simple interface built with HTML and a minimal amount of CSS to display the product data and the calculated total.

Testing Requirements

  • Total Calculation Test:
    • Test Case 1.1: Given a complete shopping cart dataset, the application must correctly calculate the total cost.
    • Expected Result: A single numeric value representing the total cost is displayed accurately.
